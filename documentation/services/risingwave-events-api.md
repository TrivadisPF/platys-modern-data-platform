# RisingWave Events API

HTTP Events API for ingesting events into RisingWave

**[Documentation](https://github.com/risingwavelabs/events-api)**
 | **[GitHub](https://github.com/risingwavelabs/events-api)**

## How to enable?

```
platys init --enable-services RISINGWAVE,RISINGWAVE_EVENTS_API
platys gen
```
