# Puppy Graph

Query your relational data as a graph in real-time. Zero ETL.

**[Website](https://www.puppygraph.com//)** | **[Documentation](https://docs.puppygraph.com)** | **[GitHub](https://github.com/puppygraph)**

## How to enable?

```
platys init --enable-services PUPPYGRAPH
platys gen
```

## How to use it?

Navigate to <http://dataplatform:28288>.
