# Apache Spark

**[Website](http://https://spark.apache.org/)** | **[Documentation](https://spark.apache.org/docs/latest/)** | **[GitHub](https://github.com/apache/spark)**

## How to enable?

```bash
platys init --enable-services SPARK
platys gen
```
