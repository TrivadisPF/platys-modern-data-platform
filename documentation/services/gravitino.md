# Gravitino

World's most powerful open data catalog for building a high-performance, geo-distributed and federated metadata lake.

**[Website](https://gravitino.apache.org/)** | **[Documentation](https://gravitino.apache.org/docs)** | **[GitHub](https://github.com/apache/gravitino)**

## How to enable?

```
platys init --enable-services GRAVITINO
platys gen
```

## How to use it?

Navigate to <http://dataplatform:28286>.
