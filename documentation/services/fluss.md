# Apache Fluss

Apache Fluss is a streaming storage built for real-time analytics.

**[Website](https://fluss.apache.org/)** | **[Documentation](https://fluss.apache.org/docs/)** | **[GitHub](https://github.com/apache/fluss)**

## How to enable?

```bash
platys init --enable-services FLUSS
platys gen
```
