# Kaskade

Kafka text user interface in style! 

**[Documentation](https://https://github.com/sauljabin/kaskade)** | **[GitHub](https://github.com/sauljabin/kaskade)**

## How to enable?

```
platys init --enable-services KASKADE
platys gen
```

## How to use it?

```
docker exec -ti kaskade kaskade
```
