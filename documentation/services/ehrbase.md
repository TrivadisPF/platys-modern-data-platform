# EHRBase

An open source openEHR server. 

**[Website](http://ehrbase.org/)** | **[Documentation](https://docs.ehrbase.org/)** | **[GitHub](https://github.com/ehrbase/ehrbase/tree/develop)**

## How to enable?

```
platys init --enable-services EHRBASE
platys gen
```

## How to use it?

Navigate to <http://dataplatform:28282>.
