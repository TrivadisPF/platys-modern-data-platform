# XTDB

An immutable SQL database for application development, time-travel reporting and data compliance. 

**[Website](https://xtdb.com/)** | **[Documentation](https://docs.xtdb.com/)** | **[GitHub](https://github.com/xtdb/xtdb)**

## How to enable?

```
platys init --enable-services XTDB
platys gen
```
