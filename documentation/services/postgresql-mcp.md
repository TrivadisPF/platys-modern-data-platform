# PostgreSQL MCP Server 

Postgres MCP Pro provides configurable read/write access and performance analysis for you and your AI agents.

 **[Documentation](https://github.com/crystaldba/postgres-mcp)** | **[GitHub](https://github.com/crystaldba/postgres-mcp)**

## How to enable?

```
platys init --enable-services POSTGRESQL, POSTGRESQL_MCP
platys gen
```

## How to use it?

Configure  to <http://dataplatform:28404/sse> if transport configuration setting is left to default.
