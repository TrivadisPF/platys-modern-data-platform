# Weaviate Verba

**[Documentation](https://github.com/weaviate/Verba?tab=readme-ov-file#verba-walkthrough)** | **[GitHub](https://github.com/weaviate/Verba)**

## How to enable?

```
platys init --enable-services VERBA
platys gen
```
