# RisingWave MCP Server

Model Context Protocol server for RisingWave.

**[Documentation](https://github.com/risingwavelabs/risingwave-mcp)** | **[GitHub](https://github.com/risingwavelabs/risingwave-mcp-server)**

## How to enable?

```bash
platys init --enable-services RISINGWAVE RISINGWAVE_MCP
platys gen
```

## How to use it?

Configure your MCP client to connect to <http://dataplatform:28409/mcp>.
