# Konga

An open source tool that enables you to manage your Kong API Gateway with ease.

**[Website](https://pantsel.github.io/konga/)** | **[Documentation](https://github.com/pantsel/konga/blob/master/README.md)** | **[GitHub](https://github.com/pantsel/konga)**

## How to enable?

```
platys init --enable-services KONG, KONGA
platys gen
```

## How to use it?

Navigate to <http://dataplatform:1337>
