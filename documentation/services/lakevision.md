# Lakevision

Lakevision is a tool which provides insights into your Apache Iceberg based Data Lakehouse. 

**[Documentation](https://deepwiki.com/lakevision-project/lakevision)** | **[GitHub](https://github.com/lakevision-project/lakevision)**

## How to enable?

```bash
platys init --enable-services LAKEVISION
platys gen
```
