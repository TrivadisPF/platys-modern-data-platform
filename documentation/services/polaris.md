# Polaris

Apache Polaris, the interoperable, open source catalog for Apache Iceberg.

**[Website](https://polaris.apache.org/)** | **[Documentation](https://polaris.apache.org/in-dev/unreleased/)** | **[GitHub](https://github.com/apache/polaris)**

## How to enable?

```
platys init --enable-services POLARIS
platys gen
```

## How to use it?

Navigate to <http://dataplatform:28284>.
