# AutoMQ

AutoMQ is a diskless Kafka® on S3. 10x Cost-Effective. No Cross-AZ Traffic Cost. Autoscale in seconds. Single-digit ms latency. Multi-AZ Availability.

**[Website](https://www.automq.com/)** | **[Documentation](https://docs.automq.com/)** | **[GitHub](https://github.com/automq/automq)**

## How to enable?

```bash
platys init --enable-services AUTOMQ
platys gen
```
