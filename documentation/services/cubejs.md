# Cube.js

Cube is an open-source headless Business Intelligence platform that connects to any data source, provides a semantic layer for metrics, and delivers data to any frontend via APIs (REST, GraphQL, SQL). It is designed to work as a middleware between your data warehouse and your analytics applications.

**[Website](https://cube.dev/)** | **[Documentation](https://docs.cube.dev/docs/introduction)** | **[GitHub](https://github.com/cube-js/cube)**

## How to enable?

```
platys init --enable-services CUBEJS
platys gen
```

## How to use it?

In dev mode, navigate to the Developer Playground at <http://dataplatform:4003>.

The Cube.js REST API is available at <http://dataplatform:4003/cubejs-api/v1>.
