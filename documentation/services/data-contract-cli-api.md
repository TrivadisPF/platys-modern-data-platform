# Data Contract CLI API

A REST API for data contract testing, linting, and exports.

**[Documentation](https://cli.datacontract.com/API)** | **[GitHub](https://github.com/datacontract/datacontract-cli)**

## How to enable?

```bash
platys init --enable-services DATA_CONTRACT_CLI_API
platys gen
```

## How to use it?

Navigate to <http://dataplatform:4242>
